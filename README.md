# Amazing LLM Apps

A curated collection of production-ready **LLM applications** built with RAG, AI Agents, Multi-agent Teams, MCP, and Voice Agents. Supports **OpenAI**, **Anthropic**, **Google**, **xAI**, **Qwen**, and **Llama** (local).

> Based on [awesome-llm-apps](https://github.com/Shubhamsaboo/awesome-llm-apps) by Shubham Saboo (Apache 2.0)
